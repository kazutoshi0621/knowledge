## ツール

### Cargo

#### プロジェクト管理
```powershell
$ cargo init                // パッケージ作成
$ cargo new hoge            // クレート作成
```

#### ビルド
```powershell
$ cargo check               // コンパイルチェック
$ cargo biuld               // ビルド（デバッグ）
$ cargo build --release     // ビルド（リリース）
$ cargo run                 // ビルド＆実行
```

`Cargo.toml`に使用したいパッケージを記述

## プロジェクト構成
### ワークスペース、パッケージ、クレート、モジュール
上にあるほど大きい単位

| 名称           | 説明                                                                           |
| -------------- | ------------------------------------------------------------------------------ |
| ワークスペース | 複数のパッケージから構成される。複数のライブラリクレートを持ちたいときに使う。 |
| パッケージ     | 複数個のバイナリクレートおよび1個以下のライブラリクレートから構成される        |
| クレート       | 複数のモジュールから構成される                                                 |
| モジュール     | 1ファイル                                                                      |
